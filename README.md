# Rust from Zero: Build a Tiny Trading Engine

A beginner-from-zero Rust project — assumes no Rust. Each step introduces a couple of files and builds cumulatively on the last, while a tutor (and a curious simulated student) walk you through every concept: Cargo, types, structs, enums, ownership, Vec, Option, pattern matching, and a tiny order-book matching engine. A read-only code walkthrough — you watch the project grow and answer understanding checks in chat.


## Stack
- Rust
- Cargo
- Ownership
- Enums
- Order book
